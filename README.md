<h1 align="center">JS FizzBuzz</h1>

<p>
  Implementazione in JavaScript del classico esercizio FizzBuzz, che sostituisce alcuni numeri
  con le parole “Fizz”, “Buzz” e “FizzBuzz” in base alla loro divisibilità.
</p>

## Obiettivo

- Generare i numeri da 1 a 100.
- Sostituire il numero con:
  - <strong>Fizz</strong> se è multiplo di 3;
  - <strong>Buzz</strong> se è multiplo di 5;
  - <strong>FizzBuzz</strong> se è multiplo sia di 3 che di 5;
  - altrimenti mostrare il numero stesso.

L’output viene stampato in console utilizzando <code>console.log()</code>.

## Tecnologie utilizzate

- JavaScript  
- DevTools del browser (console)
